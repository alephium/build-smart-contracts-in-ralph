# Companion Code For Build Smart Conracts with Ralph

## Install

```
npm install
```

## Start a local devnet for testing and development

Please refer to the documentation here: [Devnet Guide](https://wiki.alephium.org/full-node/getting-started#devnet)

## Compile

Compile the TypeScript files into JavaScript:

```
npm run compile
```

## Run Examples

e.g.

```bash
$ npx ts-node src/test-cryptograph-functions.ts
```

## Testing

```
npm run test
```
